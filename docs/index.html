FAQ, but make it chatty

Is this on the App Store?

Not yet. This is a Developer ID signed & notarized direct download. Store needs a sandboxed edition later.

Does Buddy steal my code?

No Buddy server. It reads local Grok files under ~/.grok/ and uses localhost hooks. Grok CLI still talks to xAI as usual.

Why is the page rainbow?

That’s the same mood palette as the real pet. When Buddy’s “working,” the whole site goes green. Fun on purpose.

Reply / no cursor?

Click the dark input so the window takes focus. Allow Automation for Terminal if macOS asks.

Adopt the pet.

Signed. Notarized. Colorful. Slightly unhinged in a good way.
