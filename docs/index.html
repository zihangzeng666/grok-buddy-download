FAQ

Is this the Mac App Store build?

No — this is a Developer ID signed & notarized direct download. App Store needs a sandboxed edition later.

Does Buddy send my code to a server?

Buddy reads local Grok session files under ~/.grok/ and talks to localhost for hooks. Your project work still goes through Grok CLI / xAI as usual.

macOS blocks the app?

This release is notarized. If something’s off, right-click → Open once, or re-download from the official release link.

Reply doesn’t type / no cursor?

Click the dark input field so Buddy becomes the key window. Allow Automation for Terminal/iTerm if macOS asks when sending.
